# beautyPiIV

Um sistema web desenvolvido em Django para gerenciamento de barbearias, com controle de agendamentos, finanças, estoque, perfis de usuário etc.

## Visão Geral

Este projeto foi desenvolvido como parte do projeto integrador III da Univesp e adaptado para o PI IV e inclui as principais funcionalidades para uma barbearia, tais como:

- Cadastro e autenticação de usuários (clientes / salão / administração)  
- Agendamento de serviços  
- Controle de estoque de produtos  
- Módulo de finanças (receitas, despesas)  
- Perfil de usuário com histórico  
- Layout responsivo com templates HTML + CSS + JavaScript  

## Tecnologias Utilizadas

- Python  
- Django  
- HTML / CSS / JavaScript  
- (Se houver) Banco de dados: SQLite ou MySql
